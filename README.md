# VertexEvents

Premium event rental website for **Vertex Events** — built with HTML, CSS & JavaScript, integrated with [Goodshuffle Pro](https://pro.goodshuffle.com) for live inventory management.

## Pages
- `index.html` — Homepage
- `inventory.html` — Browse Rentals (Goodshuffle Pro powered)
- `services.html` — Services
- `contact.html` — Contact / Get a Quote

## Goodshuffle Integration
Uses the Goodshuffle Pro Web Components SDK. The vendor key is embedded on every page. Inventory loads live when hosted on any web server.

## Deployment
Deploy as a static site — drag & drop to [Netlify Drop](https://app.netlify.com/drop) or use GitHub Pages.
